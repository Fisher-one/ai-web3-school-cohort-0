## 单笔交易流转

```
钱包签名 → 节点网络传播 → 内存池排队 → 构建者排序 → 验证者打包出块 → 区块上链可查询
```

简化：身份认证 → 交易授权 → 全网传播 → 排序 → 执行 → 区块确认

## 私钥 / 助记词 / 地址

| 概念 | 说明 |
|------|------|
| 私钥 | 账户最高控制权，持有即可操控账户资产 |
| 助记词 | 私钥的便捷备份，可衍生多个链上账户 |
| 地址 | 公钥换算而来，公开收款地址，可被链上溯源 |

## 钱包真实作用

- 资产不在钱包内，全部存于链上账本
- 钱包仅保管密钥、构建交易、完成签名
- 区块浏览器可查询链上余额、交易、合约数据

## 私钥 = 个人数字主权

- 无需平台注册，随机生成密钥即拥有链上账户
- 脱离传统金融身份绑定、机构管控限制
- 自主掌控资产收发与管理权限

## 私钥安全守则

1. 私钥丢失无法找回重置，泄露直接丢失资产
2. 区块链只认签名，不认实名身份
3. 被盗主因：助记词泄露、钓鱼授权、设备中毒
4. 严禁截图、网盘保存、转发私钥/助记词，慎用剪贴板
