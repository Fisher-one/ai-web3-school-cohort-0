# Handbook Feedback

在学习 [AI × Web3 School Handbook](https://aiweb3.school/zh/handbook/) 过程中遇到的问题、卡点、建议，统一记录在这里。

## 为什么记录

- 让 Handbook 更好：自己踩过的坑，可能也是其他人的坑
- 结构化复盘：写清楚卡在哪、怎么想的、建议怎么改，本身就是深度学习
- 开源贡献入口：这个目录的内容可以直接提交给 Handbook 维护者

## 每条 feedback 的格式

参考 [feedback-template.md](./feedback-template.md)，包含：

- Handbook 页面链接
- 问题描述（我理解的 vs 实际应该是什么）
- 建议改法（可选）
- 发现日期

## 文件命名

`YYYY-MM-DD-<简短关键词>.md`

例如：`2026-05-19-context-memory-auth.md`

## 提交到 Handbook

如果认为某条 feedback 有价值，可以：
1. 整理成简洁描述
2. 提交到 Handbook GitHub：https://github.com/aiweb3school/handbook（如有）
3. 或发到社群 Telegram 反馈
